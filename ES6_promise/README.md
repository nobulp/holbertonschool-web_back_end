# ES6 Promises

This project covers asynchronous programming in JavaScript using ES6 Promises.

## Requirements

- Ubuntu 20.04 LTS
- NodeJS 20.x.x
- npm 9.x.x

## Setup

Install dependencies:

```bash
npm install
```

## Run

```bash
npm run dev <filename>
```

## Tests

```bash
npm run test
```

## Lint

```bash
npm run lint
```

## Full test

```bash
npm run full-test
```
