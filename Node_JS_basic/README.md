# Node.js Basics

This project covers the basics of Node.js: reading files, handling HTTP requests, and using Express.js.

## Requirements

- Ubuntu 20.04 LTS
- NodeJS 20.x.x
- npm 9.x.x

## Setup

Install dependencies:

```bash
npm install
```

## Run

```bash
node <filename>
```

## Tests

```bash
npm run test
```

## Full test

```bash
npm run full-test
```
