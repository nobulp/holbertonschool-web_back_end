# Pagination

This project covers pagination techniques in Python, including simple helper functions, hypermedia pagination, and deletion-resilient pagination.

## Requirements

- Python 3.9+
- All files use `#!/usr/bin/env python3`
- Code follows `pycodestyle` (2.5)

## Tasks

### 0. Simple helper function
**File:** `0-simple_helper_function.py`

Function `index_range(page, page_size)` that returns a tuple `(start_index, end_index)` corresponding to the range of indexes for the given pagination parameters. Pages are 1-indexed.
