# Python Async Function

This project introduces asynchronous programming in Python 3 using `async`/`await` syntax and the `asyncio` module.

## Requirements

- Python 3.9+
- All files use `#!/usr/bin/env python3`
- Code follows `pycodestyle` (2.5)

## Tasks

### 0. The basics of async
**File:** `0-basic_async_syntax.py`

Asynchronous coroutine `wait_random(max_delay: int = 10) -> float` that waits for a random delay between 0 and `max_delay` seconds (float) and returns it.
